# Moonrock

Ett React-projekt byggt med Vite och TypeScript.

## Kom igång

1. **Installera beroenden** (kräver Node.js):
   ```bash
   npm install
   ```

2. **Starta utvecklingsservern**:
   ```bash
   npm run dev
   ```

3. Öppna [http://localhost:5173](http://localhost:5173) i webbläsaren.

## Skript

| Kommando | Beskrivning |
|----------|-------------|
| `npm run dev` | Startar utvecklingsservern |
| `npm run build` | Bygger för produktion |
| `npm run preview` | Förhandsgranskar produktionsbyggen |
| `npm run lint` | Kör ESLint |

## Teknisk stack

- **React 18** – UI-bibliotek
- **Vite 5** – Byggverktyg
- **TypeScript** – Typsäkerhet
